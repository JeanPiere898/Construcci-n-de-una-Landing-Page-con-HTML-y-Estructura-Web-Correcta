
## Características

- **HTML5 semántico**: Uso de etiquetas como `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- **Responsive design**: Adaptable a diferentes tamaños de pantalla
- **Accesibilidad**: Atributos ARIA y estructura semántica para mejor accesibilidad
- **Interactividad**: JavaScript para funcionalidades básicas
- **Buenas prácticas**: Código limpio, comentado y organizado

## Secciones de la landing page

1. **Header**: Logo y eslogan de la empresa
2. **Navegación**: Menú principal con enlaces a las secciones
3. **Hero**: Sección principal con llamado a la acción
4. **Servicios**: Tarjetas con información de servicios ofrecidos
5. **Nosotros**: Información sobre la empresa
6. **Testimonios**: Comentarios de clientes satisfechos
7. **Contacto**: Formulario de contacto e información
8. **Footer**: Enlaces adicionales y derechos de autor

## Cómo usar

1. Clona o descarga este proyecto
2. Abre el archivo `index.html` en tu navegador
3. Personaliza el contenido según tus necesidades

## Tecnologías utilizadas

- HTML5
- CSS3 (con variables CSS y Flexbox/Grid)
- JavaScript (ES6)

## Buenas prácticas implementadas

- Uso de etiquetas semánticas de HTML5
- Estructura de carpetas organizada
- Código comentado y legible
- Formularios accesibles
- Navegación por teclado compatible
- Imágenes con texto alternativo
- Metaetiquetas para SEO básico
- Favicon incluido

## Despliegue en GitHub Pages

1. Crea un repositorio en GitHub
2. Sube todos los archivos del proyecto
3. Ve a Settings > Pages
4. Selecciona la rama principal (main/master) y guarda
5. Tu landing page estará disponible en `https://[tu-usuario].github.io/[nombre-repositorio]/`
